# HTML Tags Reference

This document lists common HTML tags, their full spelling or origin (if applicable),
and their purpose.  
Note: Many HTML tags are not true acronyms; they are descriptive words or historical abbreviations.

---

## Document Structure

| Tag | Full spelling / origin | Description |
|-----|------------------------|-------------|
| <!DOCTYPE> | Document Type Declaration | Declares HTML version (HTML5) |
| html | HyperText Markup Language | Root element of an HTML document |
| head | Head section | Contains metadata |
| body | Body section | Contains visible page content |

## Notes

- HTML tags are primarily semantic, not acronym-based.
- Older tags tend to use abbreviations.
- Newer HTML5 tags use real English words.
- Use semantic tags for accessibility, SEO, and maintainability.
